QUICK START:
================================================================================

1. Validate setup:
   ./validate-setup.sh

2. First build:
   make rebuild

3. Start:
   make start

4. Access dashboard:
   https://localhost:8443/dashboard

COMMON WORKFLOWS:
================================================================================

Frontend changes not showing:
  make clean-frontend
  cd frontend && npm install && npm run build
  make build
  make restart

Backend changes not showing:
  make clean-docker
  make build
  make restart

Complete fresh start:
  make clean-all
  make rebuild
  make start

After git pull:
  make rebuild
  make start
